How To Connect


1. Open up MCPE 0.14

2. Click on "New"

3. Press the weird looking button next to "Advanced"

4. Enter the designated IP Address for the server you would like to connect to

5. Change the port to the port of the server you would like to connect to

6. Click on "Add Server"

7. Go to the homescreen

8. Click on the server!

9. Register by typing your desired password and follow the instructions shown in chat, OR Login by typing the password that you registered with.



How To Make Plugins


1. Make sure that all php files start with: "?php"

2. Next Insert Your namespace using the format of Format = (AuthorName)\(Pluginname): "namespace SQPE\Plugin;"

3. Enter "use" statements, such as: "use pocketmine\plugin\PluginBase;"

4. Type the class: "class Main extends PluginBase{" The format should be "class (filename excluding '.php') extends PluginBase{"

So far, you have typed

?php
namespace SQPE\Plugin
use pocketmine\plugin\PluginBase;
class Main extends PluginBase{


You must be so proud of yourself!

5. Type the loading messages: "public function onLoad(){" (then hit enter, and type) "$this->getLogger()->info("The Plugin is Loading...");"

6. You can do a similar thing with enabling and disabling (public function onEnable(){ $this->getLogger()->info("The Plugin has been Enabled!")) and (public function on Disable(){ $this->getLogger()->info("The Plugin has been Disabled!"))

So far, you have typed

?php

use pocketmine\plugin\PluginBase;

I hope you like this so far!

7. Now we are going to change some of the earlier coding to allow more functionality (look at example below step ) This allows color, and many more features.

8. Now, we add some code that will change the message when someone joins the server. " public function onJoin(PlayerJoinEvent $event){ $player = $event->getPlayer(); $name = $player->getName(); $event->setJoinMessage(TextFormat::ORANGE."$name Joined The Amazing Server! Yay!");"

So far, you have typed

?php
namespace SQPE\plugin;
use pocketmine\plugin\PluginBase;
use pocketmine\event\Listener;
use pocketmine\event\player\PlayerJoinEvent;
use pocketmine\utils\TextFormat;
class Main extends PluginBase implements Listener{
public function onLoad(){
$this->getLogger()->info("The Plugin is Loading...");
}
public function onEnable(){
$this->getServer()->getPluginManager()->registerEvents($this,$this);
$this->getLogger()->info("The Plugin has been Enabled!");
}
public function onDisable(){
$this->getLogger()->info("The Plugin has been Disabled!");
}
public function onJoin(PlayerJoinEvent $event){
$player = $event->getPlayer();
$name = $player->getName();
$this->getServer()->broadcastMessage(TextFormat::ORANGE."$name Joined The Amazing Server! Yay!");
}
Good job! This is a lot of code! Not enough, though...
