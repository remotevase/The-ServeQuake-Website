How To Connect


1. Open up MCPE 0.14

2. Click on "New"

3. Press the weird looking button next to "Advanced"

4. Enter the designated IP Address for the server you would like to connect to

5. Change the port to the port of the server you would like to connect to

6. Click on "Add Server"

7. Go to the homescreen

8. Click on the server!

9. Register by typing your desired password and follow the instructions shown in chat, OR Login by typing the password that you registered with.



How To Make Plugins


1. Make sure that all php files start with: "

2. Next Insert Your namespace using the format of Format = (AuthorName)\(Pluginname): "namespace SQPE\Plugin;"

3. Enter "use" statements, such as: "use pocketmine\plugin\PluginBase;"

4. Type the class: "class Main extends PluginBase{" The format should be "class (filename excluding '.php') extends PluginBase{"
So far, you have typed
" namespace SQPE\Plugin
use pocketmine\plugin\PluginBase;
class Main extends PluginBase{"
